網站流量、用戶行為、轉換率 — 必要串接

📋 如何取得以下資訊（依序操作）
  1. 前往 analytics.google.com
  2. 點擊左下角「管理（齒輪）」→「資料串流」→ 點選你的網站
  3. 複製右上角的 評估 ID（格式：G-XXXXXXXXXX）
  4. 前往 Google Cloud Console → 建立或選擇專案
  5. 「API 與服務」→ 啟用「Google Analytics Data API」
  6. 「IAM 與管理」→「服務帳戶」→ 建立服務帳戶 → 下載 JSON 金鑰
  7. 回到 GA4 後台「管理」→「帳戶存取管理」→ 加入服務帳戶 Email（查看者權限）
在 GA4「管理」→「資料串流」→ 點入你的網站 → 右上角「評估 ID」

點擊或拖曳 JSON 金鑰檔案到這裡

從 Google Cloud Console 下載的 service-account-xxxx.json

或直接貼上 JSON 內容
JSON 金鑰包含 private_key 欄位，這是存取 GA4 的憑證，請勿分享給他人。
Google Search Console
關鍵字排名、自然點擊、搜尋曝光
待完成
📋 如何取得以下資訊
  1. 前往 search.google.com/search-console
  2. 確認你的網站已驗證（左側能看到你的 domain）
  3. 複製你的網站 URL（例：https://yoursite.com 或 sc-domain:yoursite.com）
  4. 在 Search Console「設定」→「使用者與權限」→ 加入 Service Account Email（與 GA4 相同金鑰即可）
與你在 Search Console 驗證的網站 URL 完全一致（注意有沒有 / 結尾）
💡 小提示：Search Console 使用與 GA4 相同的 Service Account JSON 金鑰，上面填好 GA4 金鑰後這裡不需要再上傳。

Meta Ads（Facebook / Instagram）選填
FB/IG 廣告花費、ROAS、受眾觸及數據
選填
📋 如何取得以下資訊（這是最複雜的步驟，約 10 分鐘）
  1. 前往 business.facebook.com → 「設定」→「廣告帳號」
  2. 複製你的廣告帳號 ID（格式：act_XXXXXXXXXXXXXXX）
  3. 「設定」→「系統用戶」→「新增系統用戶」（角色：員工）
  4. 點擊「新增資產」→ 選擇你的廣告帳號 → 授予「廣告帳號分析師」權限
  5. 點擊「產生新 Token」→ 選擇 App → 勾選「ads_read」權限 → 複製 Token
在 Meta 商業管理平台「廣告帳號」頁面找到，格式為 act_123456789012345
Meta 系統用戶的長效存取權杖（Long-lived Access Token），有效期約 60 天需定期更新。

📋 如何取得 OpenAI API Key
  1. 前往 platform.openai.com/api-keys
  2. 點擊「Create new secret key」→ 複製金鑰（格式：sk-xxxx）
  3. 確認帳號有足夠的 Credits（每次 AI 分析約消耗 $0.001–$0.005）
此 Key 僅儲存在你的本機伺服器，不會上傳至任何雲端服務。
