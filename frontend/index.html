      新手指南 — 10 分鐘完成串接
      
        
      
      
      
      
        
      
          不需要技術背景，按照以下順序操作，10 分鐘內即可看到你的真實數據。
        
      
        
        
          
      1. 確認你有 Google Analytics 帳號
        前往 analytics.google.com，確認你的網站已安裝 GA4（新版）。若還在用舊版 Universal Analytics，需先升級。
        預計 5 分鐘
        2. 
          
      2. 點擊右上角「串接資料源」
        選擇你要串接的平台，系統會帶你進入逐步教學，每一步都有截圖說明。
        預計 2 分鐘
        3. 
          
      3. 貼上你的 Property ID 或金鑰
        GA4 評估 ID 格式像 G-XXXXXXXXXX，在 GA4 後台「管理」→「資料串流」裡找得到。
        不需要工程師
        4. 
          
      4. 數據自動載入 Dashboard
        串接完成後，頁面會自動抓取你的真實數據，AI 也會開始分析並給出本週建議。
        自動完成
        5.
