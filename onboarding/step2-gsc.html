Search Console API 授權步驟

GSC 數據將在 Phase 2 正式上線，現可提前完成授權設定。

  1. 啟用 Search Console API

    在同一個 GCP 專案中，搜尋並啟用 Google Search Console API。

  2. 使用相同的 Service Account

    Step 1 建立的 Service Account 可直接使用，無需額外建立。

  3. 在 Search Console 中授權

    Search Console → 設定 → 使用者和權限 → 新增使用者 → 貼上 Service Account email → 權限選「完整」。

  4. 設定網站 URL

    在 .env 中加入：
    GSC_SITE_URL=https://yourdomain.com

📊 上線後可看到的數據 搜尋關鍵字排名、曝光次數、點擊率 (CTR)、平均排名位置。
