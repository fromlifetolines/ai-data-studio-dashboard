Meta Marketing API 授權步驟

Meta Ads 數據將在 Phase 3 正式上線，完成後即可實現全渠道 ROAS 分析。

  1. 建立 Meta 開發者應用程式

    前往 Meta for Developers → 建立應用程式 → 類型選「Business」。

  2. 新增 Marketing API 產品

    在應用程式 Dashboard 中，新增「Marketing API」產品。

  3. 取得 Access Token

    使用 OAuth 流程或 System User Token 取得長效 Access Token。
    在 .env 中加入：
    META_ACCESS_TOKEN=your_token
    META_AD_ACCOUNT_ID=act_123456789

  4. 確認全渠道 Dashboard

    完成全部 4 步驟後，Dashboard 將顯示 GA4 + Google Ads + Meta Ads 的整合 ROAS 與 AI 跨渠道建議。

🎉 恭喜！完成全部串接 你現在擁有完整的全渠道數據分析能力。AI 將自動比較各渠道 ROAS，提供預算分配建議。
