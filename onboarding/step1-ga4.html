Google Analytics 4 授權步驟

完成以下 5 個步驟，即可在 Dashboard 看到真實 GA4 數據。

  1. 建立 GCP 專案

    前往 Google Cloud Console，建立新專案（或使用現有專案）。

  2. 啟用 Google Analytics Data API

    在 GCP 專案中，搜尋並啟用 Google Analytics Data API。

  3. 建立 Service Account 金鑰

    IAM → Service Accounts → 建立帳戶 → 金鑰 → 新增 JSON 金鑰。
    將下載的 JSON 檔重新命名為 service-account.json，放入專案的 credentials/ 資料夾。

  4. 在 GA4 中授權 Service Account

    GA4 管理 → 資源存取管理 → 新增使用者 → 貼上 Service Account 的 email（格式：xxx@xxx.iam.gserviceaccount.com）→ 角色選「檢視者」。

  5. 設定 Property ID

    GA4 管理 → 資源設定 → 複製「資源 ID」（純數字）。
    在專案根目錄建立 .env 檔案，加入：
    GA4_PROPERTY_ID=你的資源ID

💡 小提示 完成後重新啟動後端伺服器，Dashboard 右上角的狀態會從「Demo Mode」變成「Live Data」。
