Google Ads API 授權步驟

Google Ads 數據將在 Phase 3 正式上線。

  1. 申請 Google Ads API 存取權

    前往 Google Ads API 文件，申請 Developer Token。

  2. 建立 OAuth 2.0 憑證

    GCP Console → API 和服務 → 憑證 → 建立 OAuth 2.0 用戶端 ID（Web 應用程式）。

  3. 取得 Refresh Token

    使用 OAuth 流程取得 Refresh Token，存入 .env：
    GOOGLE_ADS_CUSTOMER_ID=1234567890
    GOOGLE_ADS_REFRESH_TOKEN=your_token

  4. 確認 Dashboard 顯示

    完成後，Dashboard 的 Google Ads 卡片會從 mock 數據切換為真實投放數據。

💰 上線後可看到的數據 各 Campaign 花費、點擊、轉換、CPA、ROAS，以及每日趨勢圖。
